## 模拟简历展示系统  

### 功能
  * express搭建后端接口，提供登录注册，简历的增删查改，点赞数及评论记录功能  
  * mongodb Atlas作为云端数据库
  * vue使用自行搭建的组件，vuex管理用户的登录状态，请求状态及获取到的用户信息
  * 进行token验证，通过路由跳转及路由守卫展示不同页面  
  
   不足：本地测试有效，但没有发布上线 —— [本地展示效果](./result-show)
